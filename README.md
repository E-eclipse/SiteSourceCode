# 🌒 E-eclipse | Портфолио

Современный, адаптивный сайт-портфолио с анимированным интерфейсом и интерактивными элементами. ✨

## 🌟 Особенности

- 🌑 Анимированная загрузочная страница с эффектом затмения
- ⭐ Интерактивный фон с частицами (particles.js)
- 🎭 Плавные анимации при прокрутке
- 📱 Адаптивный дизайн для всех устройств
- 📊 Таймлайн проектов с анимированными точками
- 🎨 Градиентные эффекты и современные визуальные элементы
- ⌨️ Анимированный текст с эффектом печатающейся машинки
- 🎯 Кастомный favicon с изображением солнечного затмения

## 🛠 Технологии

- 🌐 HTML5
- 🎨 CSS3 (с использованием современных возможностей)
- ⚡ JavaScript (ES6+)
- 🖼️ SVG для создания favicon
- 📚 Библиотеки:
  - ✨ AOS (Animate On Scroll)
  - 🌠 Particles.js

## 📦 Структура проекта

```
├── 📄 index.html         # Основной HTML файл
├── 🎨 styles.css         # CSS стили
├── ⚡ script.js          # JavaScript код
├── 🎯 favicon.png        # Иконка сайта
├── 🎨 favicon.svg        # Исходный SVG файл иконки
└── 📖 README.md         # Документация проекта
```

## 🚀 Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/E-eclipse/portfolio.git
```

2. Откройте `index.html` в вашем браузере 🌐

## 💻 Требования

- 🌐 Современный веб-браузер с поддержкой CSS3 и ES6
- ⚡ Включенный JavaScript
- 📡 Стабильное интернет-соединение для загрузки внешних библиотек

## 🎨 Цветовая схема

- 🟠 Основной: `#ff7b00` (Оранжевый)
- 🔸 Вторичный: `#ff4800` (Темно-оранжевый)
- ⭐ Акцент: `#ffd700` (Желтый)
- ⚫ Темный: `#121212`
- ⚪ Светлый: `#ffffff`

## 📱 Адаптивность

Сайт адаптирован под следующие разрешения:
- 🖥️ Десктоп (> 1024px)
- 📱 Планшет (768px - 1024px)
- 📱 Мобильные устройства (< 768px)

## 🔧 Кастомизация

Для изменения цветовой схемы отредактируйте переменные в начале файла `styles.css`:

```css
:root {
    --primary: #ff7b00;    /* 🟠 Оранжевый */
    --secondary: #ff4800;  /* 🔸 Темно-оранжевый */
    --accent: #ffd700;    /* ⭐ Желтый */
    --dark: #121212;      /* ⚫ Темный */
    --light: #ffffff;     /* ⚪ Светлый */
}
```
